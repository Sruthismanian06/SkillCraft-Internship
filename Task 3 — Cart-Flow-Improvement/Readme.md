# Task 3 — Cart Flow Improvement (E-commerce)

## 🎯 Objective
Improve cart and product detail flow to enhance purchase clarity and reduce user drop-off.

---

## 🧠 UX Problem
- Confusing cart layout  
- Lack of clarity in pricing and product details  
- High drop-off during checkout process  
- Poor decision-making support for users  

---

## 🎨 Solution
- Redesigned cart layout for better clarity  
- Improved product detail visibility  
- Simplified checkout flow structure  
- Enhanced user decision-making experience
