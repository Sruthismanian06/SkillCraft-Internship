# Task 2 — News App UI Design

## 🎯 Objective
Design a clean and readable news application interface focused on content hierarchy and user engagement.

---

## 🧠 UX Problem
- Information overload in news feeds  
- Poor readability due to weak typography  
- Lack of clear content hierarchy  
- Low engagement due to cluttered UI  

---

## 🎨 Solution
- Designed structured layout for better content scanning  
- Improved typography for readability and clarity  
- Created clean card-based UI for news articles  
- Enhanced visual hierarchy for better engagement  

---

## 🛠️ UX Skills Applied
Information Architecture · Visual Hierarchy · UI Design · Content Structuring · Readability Optimization
