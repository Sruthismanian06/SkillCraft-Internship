# Task 4 — Fitness App User Journey Design

## 🎯 Objective
Design a smooth onboarding user journey for a fitness application.

---

## 👤 User Flow
Welcome Screen → Goal Selection → Fitness Preferences → Setup Completion

---

## 🧠 UX Problem
- Complex onboarding processes in fitness apps  
- User drop-off during setup  
- Lack of guided user flow  
- Confusing goal selection experience  

---

## 🎨 Solution
- Designed step-by-step onboarding flow  
- Simplified goal selection process  
- Reduced friction in user setup journey  
- Created clear and guided user experience  

---

## 🛠️ UX Skills Applied
User Journey Design · Onboarding UX · User Flow Mapping · Information Architecture · Interaction Design
